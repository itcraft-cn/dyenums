## 1. 项目概述

dyenums 是一个 Java 8+ 动态枚举库，采用 Map+Factory 模式解决 Java 静态枚举的运行时扩展问题。项目采用 Maven 多模块架构：

| 模块 | 职责 |
|------|------|
| dyenums-core | 核心接口与实现 |
| dyenums-spring | Spring 集成 |
| dyenums-config-file | 文件配置加载 |
| dyenums-config-db | 数据库配置加载 |
| dyenums-test | 测试模块 |
